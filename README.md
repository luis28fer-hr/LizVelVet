# Landing Page LIZVELVET

Archivos incluidos:

- `index.html` — estructura principal de la landing page.
- `styles.css` — estilos, responsive design, animaciones, hover effects y colores.
- `script.js` — menú móvil, carrusel, contadores, modales y formulario demo.
- `assets/` — imágenes SVG locales de ejemplo para que la página funcione sin internet.
